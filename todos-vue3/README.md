# todos-vue2
用vue3编写的Todo

## 项目创建
```
# 初始化
npm init vite-app todos-vue3
# 安装路由
npm i vue-router@next -S
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```
