# todos-vue2
用vue3编写的Todo

## Project create
```
npm init vite-app todos-vue3
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```
